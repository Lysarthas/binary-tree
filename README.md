Binary Tree
=============

[Self-balancing] [binary tree] module that uses [AVL trees].

What Problem This Solves
------------------------
* Fast insertion
* Fast retrieval
* Fast deletion
* Objects with duplicate values may be added

How This Solves It
------------------

Usage
-----
New tree

`tree = AVLTree()`

Insert Entity into the tree

`tree.insert(5, "optional unique ID")`
